# guoqi-job-hunter

2027 届国企校招自动监控与半自动投递助手，面向通信工程学生，先实现招聘公告监控、岗位筛选和 Excel 导出，后续再扩展半自动投递。

## 用户背景

* 学校：四川轻化工大学
* 专业：通信工程
* 年级：大三
* 证书：CET4、计算机二级
* 经历：供电所实习经历
* 目标：国家电网、南方电网、三大运营商、央国企技术岗

## 项目目标

帮助用户持续关注 2027 届国企校招信息，自动整理招聘公告，筛选通信工程相关岗位，生成投递优先级和可导出的岗位清单。

本项目遵守投递安全边界：不绕过验证码，不自动提交最终申请，最终投递前必须由用户人工确认。

## 功能

* 招聘公告监控：读取企业配置，抓取公开招聘入口页面并识别疑似校招公告。
* 岗位匹配评分：根据通信工程、网络运维、信息化、电力通信、云网、技术支持等关键词评分。
* 国企投递优先级排序：结合目标单位、岗位相关度、学历要求、地区和实习经历进行排序。
* Excel 导出：将岗位清单导出为 `.xlsx`，便于筛选、记录和投递跟进。
* 提醒信息生成：根据报名时间、优先级和推荐原因生成投递提醒。
* 简历优化建议：提示应突出供电所实习、通信网络课程、Office/计算机二级和 CET4。
* 面试准备建议：提示准备电力通信、计算机网络、运营商网络、实习复盘和国企价值观表达。

## 使用方法

1. 在 `config/companies.yaml` 中维护需要监控的企业和招聘入口。
2. 运行 `src/monitor.py` 抓取公开招聘公告，生成公告候选列表。
3. 使用 `src/matcher.py` 对岗位标题和公告内容进行匹配评分。
4. 使用 `src/export_excel.py` 导出岗位清单 Excel。
5. 使用 `src/notify.py` 生成提醒文本。
6. 按 `docs/投递流程.md` 完成人工确认、登录、验证码、简历上传和最终提交。

## 安全原则

* 不绕过验证码。
* 不保存网站登录密码。
* 不自动提交最终申请。
* 不替用户确认志愿、岗位或个人信息。
* 最终投递前必须人工检查公告、岗位、简历和报名信息。
